# Social media content

Drafts, copy, images, and published-asset references for APLOMB.'s social
channels go here. Currently empty — populate as launches happen.

Suggested subdir conventions when you start:

```
social-media/
├── instagram/
│   ├── posts/          ← static image posts (filename: YYYY-MM-DD-slug)
│   ├── stories/
│   └── reels/
├── tiktok/
├── x-twitter/
└── press-kit/          ← logos, founder bio, fact sheet — for media inquiries
```

Generated visuals from `design-scratch/` can be moved/copied here once
finalized for a specific channel.
